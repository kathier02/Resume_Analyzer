# Resume Skill Analyzer

A Flask-based web application that analyzes a candidate's resume against a job description and identifies missing skills that may improve the resume's match for a specific role.

## Features

* Upload resume in PDF format
* Paste job descriptions directly into a text area
* Extract text from resumes automatically
* Compare resume content with job requirements
* Identify missing technical skills
* Simple and responsive user interface

## How It Works

1. Upload a PDF resume.
2. Paste the job description into the provided text area.
3. Click **Analyze Skills**.
4. The application compares the resume against the job description.
5. Missing skills are displayed to help improve resume relevance.
